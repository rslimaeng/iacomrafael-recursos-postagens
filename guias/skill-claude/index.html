O que é

Uma Skill ensina o Claude uma vez.
E ele nunca mais esquece.

Pense assim: um prompt é uma instrução que você dá uma vez. Uma Skill é uma instrução que fica salva e o Claude carrega automaticamente quando precisar.

"Prompts são instruções que você dá uma vez. Skills são instruções que ficam para sempre."
1

O que é uma Skill?

Um conjunto de instruções salvas que ensinam o Claude a fazer uma tarefa específica do seu jeito — sempre. Você escreve uma vez, salva, e ele aplica automaticamente.

2

Por que usar?

✓ Parar de repetir contexto
✓ Manter padrão entre chats
✓ Acelerar tarefas recorrentes
✓ O Claude já te conhece antes de você falar

3

Onde funciona?

Em todos os lugares do Claude: chat normal, Claude Cowork e Code Tab. Você cria uma vez e funciona em qualquer conversa.

4

Quem pode usar?

Todos os planos: gratuito, Pro, Max, Team e Enterprise. Não precisa saber programar. Cria em conversa, sem código.

O problema que Skill resolve

Você já escreveu isso mais de uma vez?

Toda vez que abre o Claude para fazer a mesma tarefa, você repete as mesmas instruções. Skill elimina esse loop.

Sem Skill — toda semana
"Escreve em tom direto, frases curtas, sem floreio, sem introdução longa, sem 'certamente', sem opções, entrega pronto para copiar e colar..."

Você cola isso toda vez que precisa escrever algo.

Com Skill ativa
"Escreve o e-mail de follow-up para o cliente X."

O Claude já sabe o tom, o formato e o que evitar. Entrega pronto.

Como criar

Crie sua primeira Skill em 5 passos

Tudo via interface do claude.ai — sem código, sem arquivo manual, sem configuração técnica.

⚠️
Antes de começar: 1 configuração necessária
Vá em Configurações → Capacidades e ative "Execução de código e criação de arquivos". Sem isso, Skills não aparecem no menu.

    A opção "Criar com Claude" monta o arquivo automaticamente em conversa — você só descreve o que quer.

  • 3

    Descreva o que você quer (a parte mais importante)

    Uma caixa de conversa vai aparecer. Descreva sua Skill em português, como se estivesse explicando para um assistente humano: o que você quer que ele faça, como, e o que nunca fazer.

    Use o modelo dos 5 elementos da seção abaixo para estruturar melhor sua descrição.

  • 4

    Revise o que o Claude criou

    O Claude vai montar o arquivo da Skill automaticamente. Leia o que ele gerou — se algo não ficou certo, peça ajustes em conversa. Não precisa editar código manualmente.

  • 5

    Ative e teste

    Clique em Salvar e ative a Skill. Abra um novo chat e peça algo que deveria acionar ela. Se não acionar, reescreva a descrição da Skill para ser mais específica — é ela que o Claude usa para decidir quando aplicar.

O que colocar

5 elementos para uma Skill eficaz

Quanto mais específico, mais alinhado o resultado. Skill vaga entrega resultado vago.

  • 🎯
    Objetivo
    Para que serve a Skill — a tarefa específica que ela cobre.Ex: "Escrever e-mails e mensagens profissionais no meu tom"
  • ✍️
    Estilo
    Como você quer que o Claude escreva — tom, linguagem, nível de formalidade.Ex: "Direto, profissional, sem floreio, sem introdução longa"
  • 📐
    Formato
    Como entregar a resposta — lista, parágrafo, tabela, pronto para copiar.Ex: "Texto pronto para copiar e colar, sem opções, sem perguntas"
  • 🚫
    Regras
    O que o Claude nunca deve fazer nessa Skill.Ex: "Nunca usar 'certamente', nunca dar opções sem eu pedir"
  • 💡
    Exemplo
    Uma amostra do resultado bom e do resultado ruim — opcional, mas ajuda muito.Ex: mostrar um e-mail no tom certo e um no tom errado
Templates

3 templates prontos para começar

Cole qualquer um na tela "Criar com Claude" e ajuste para a sua realidade.

Template 01
Comunicação e Tom de Escrita
Para qualquer texto: e-mails, mensagens, resumos. A mais útil para começar.
Comece aqui
Quero uma Skill de comunicação profissional. Quando eu pedir para escrever qualquer texto (e-mail, mensagem, resumo, relatório), siga estas regras: Tom: direto, profissional, sem floreio Frases: curtas, ativas, máximo 2 linhas por parágrafo Evitar: "certamente", "claro que sim", "espero que este e-mail te encontre bem", introduções longas, opções desnecessárias Entrega: texto pronto para copiar, sem variações, sem perguntas adicionais Exemplo bom: "Segue o relatório conforme alinhado. Qualquer dúvida, fico à disposição." Exemplo ruim: "Olá! Espero que este e-mail te encontre muito bem. Venho por meio deste e-mail para compartilhar..."
Template 02
Resumo de Reunião
Transforma anotações bagunçadas em resumo estruturado com decisões e próximos passos.
Quero uma Skill de resumo de reunião. Quando eu colar anotações ou transcrição de uma reunião, entregue um resumo estruturado: 1. Contexto (1 parágrafo, máximo 3 linhas) 2. Principais decisões tomadas (bullet points) 3. Próximos passos (responsável + ação + prazo se mencionado) 4. Pontos em aberto (o que ficou sem definição) Formato: tópicos, nunca texto corrido Tom: objetivo, neutro, sem opiniões Tamanho: máximo 1 página A4 Se alguma informação não estiver clara nas anotações, escreva "[não mencionado]" — nunca invente.
Template 03
Análise de Documento
Para quando você cola um PDF, relatório ou texto e quer análise rápida e acionável.
Quero uma Skill de análise de documento. Quando eu compartilhar um documento (PDF, texto, tabela), entregue: 1. Resumo executivo (3–5 linhas: o que é, para quem, qual o ponto central) 2. Principais pontos (máximo 5 bullets — só o acionável) 3. Riscos ou alertas (o que merece atenção) 4. Pergunta para reflexão (1 pergunta que o documento levanta mas não responde) Tom: consultor experiente falando com gestor ocupado Evitar: repetir o óbvio, linguagem técnica desnecessária Formato: tópicos, nunca texto corrido
Para não errar

4 dicas rápidas

🎯

Uma Skill por tarefa

Não tente resolver tudo em uma. Múltiplas Skills focadas funcionam muito melhor que uma Skill genérica.

🔑

A descrição é o gatilho

O Claude usa a descrição da Skill para decidir quando aplicá-la. Se não aciona, reescreva a descrição para ser mais específica.

🧩

Skills se somam

O Claude pode usar duas Skills ao mesmo tempo sem você pedir. Uma Skill de tom + uma de formato trabalham juntas automaticamente.

🚀

Comece pela comunicação

O Template 1 é o melhor ponto de entrada. 10 minutos de configuração, resultado toda semana para qualquer texto.
